## 概述

基于 hexo 的网站

## 本地测试

```
npm install  &&  npm start
```

注：查看依赖库：

```
npm ls --depth 0

一个提示：
npm ERR! peer dep missing: @types/node@^14.11.2, required by cheerio-select-tmp@0.1.0

添加："@types/node": "^14.11.2"
```
